SpotPass Archival Project / Wii U ガイド (CFW)

前提:

  • Aroma か TiramisuがあるWiiUシステム
  • FAT32かexFATでフォーマットされたSDカード
  1. Homebrew App Storeで「Dumpling」の最新版をダウンロードします。
    • Homebrew App Storeがなければ、 GitHub もダウンロードできます
  2. HOMEメニュー、 または Homebrew Launcherで「Dumpling」を起動します。
  3. Dumplingが開いたら、DパッドとAボタンでナビゲートし、「Dump all SpotPass data」というオプションを選択します。
  4. SDカードが選択されていなければ、使いたいSDカードを選択し、「OK」を選択します。
  5. ダンプが終わると、SpotPass　Archival Projectにファイルをアップロードするか聞いてきます。[はい」を選択してください。
    • キャッシュをインターネットにアップロードできなかった場合、task.dbファイルはSDカードの/Dumpling/SpotPass Files/フォルダに保存されます。
  6. システムがファイルをアップロードできなかった場合は、下の緑色のリンクをクリックすると、ファイルを送信することもできます。
