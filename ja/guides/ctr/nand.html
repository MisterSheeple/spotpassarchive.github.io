SpotPass Archival Project / 3DS ガイド (NAND ダンプ)

警告：このガイドは上級者向けです。物理的なシステムを利用できない場合や、偶々古いNANDのバックアップがある場合の最終手段としてご利用してください。.

前提:

  • 2DS/3DS 本体のNAND
  1. 本体のNANDのダンプを取得します.
  2. Autocopy をPCにダウンロードして、pip install -r requirements.txtを実行して依存関係をインストールします。
  3. python3 autocopy.pyを使用してautocopyを実行し、画面の指示に従って、プロンプトが表示されたらNANDへのパスを入力する。
    • ガイド付きインターフェースに問題があったり、より多くのオプションが必要な場合は、通常のCLIプログラムとして使用することができます。詳細は python3 autocopy.py --help を実行してください
  4. すべてがうまくいけば、NANDがあるのと同じフォルダに partitionA.bin ができるはずです。
  5. 5. AutocopyがSpotPass Archival Projectにファイルをアップロードするか聞いてきます。「y」を押してエンターキーを押すとアップロードされます。
  6. 6. アップロードに失敗したり、ファイルをアップロードできない場合は、以下の緑色のリンクからダンプを送信してください。
