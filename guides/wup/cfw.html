SpotPass Archival Project / Wii U Guide (CFW)

Prerequisites:

  • Any Wii U running Aroma CFW
  • A network connection
  • A PC with an FTP client installed
  1. Download the latest release of the FTPiiU plugin for Aroma, available as a release on GitHub.
    • When downloading from GitHub, extract the file inside of the .zip file into the /wiiu/environments/aroma/plugins/ folder of your SD card.
  2. Load into Aroma.
  3. Hold L + D-PAD DOWN + SELECT to open the plugins menu, and select FTPiiU. This should give you your Wii U's IP address.
  4. While you're there, go into the FTPiiU settings and enable "Allow access to system files".
  5. Connect to your Wii U's IP address using an FTP client on the same network as your system.
  6. Navigate to /storage_mlc/usr/save/system/boss/.
  7. Copy all of the folders in this directory to your PC.
  8. When done, the files named task.db will be located in the folders you copied to your PC.
  9. Submit your dump using the green link below.
